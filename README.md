# 📖 Al-Quran Al-Kareem - Advanced Web App

> بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ

An advanced, feature-rich web application for reading and studying the Holy Quran with Arabic, English, and Bangla translations. Designed with modern web technologies and Islamic aesthetics for an unparalleled user experience.

## 🌟 Features

### 📚 Complete Quran Text
- **6,236 Ayahs** across **114 Surahs**
- Authentic Arabic text preserved exactly as revealed
- Complete English and Bangla translations
- Accurate surah metadata and numbering

### 🔍 Advanced Search
- **Multi-language search** (Arabic, English, Bangla)
- **Regex support** for advanced queries
- **Whole word matching**
- **Case-sensitive search** options
- **Filter by Surah** and **Revelation type** (Meccan/Medinan)

### 🎨 Modern Interface
- **Multiple themes**: Dark, Light, and Sepia
- **Responsive design** for all devices
- **Beautiful Islamic-inspired styling** with gold accents
- **Smooth animations** and transitions

### 🧠 Study Tools
- **Memorization mode** - Hide text to test recall
- **Bookmarking system** for favorite ayahs
- **Progress tracking** with reading progress bars
- **Daily Ayah** feature for spiritual reflection

### ⚙️ Customization
- **Adjustable font sizes** (50%-200%)
- **Language toggles** for personalized reading
- **Settings panel** for all preferences

## 📱 Usage

### Basic Reading
1. Click **"Browse Surahs"** to explore all chapters
2. Select any surah to start reading
3. Use language toggles to show/hide translations

### Advanced Search
1. Go to **"Advanced Search"** panel
2. Enter search terms in any language
3. Use filters and options for precise results
4. Click on results to jump to specific ayahs

### Memorization
1. Open any surah in reader mode
2. Click **"🧠 Memorize"** to hide text
3. Test your memory and reveal when ready
